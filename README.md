# react-native-textinput-maxlength-fixed

fix TextInput's maxLength bug when  using a non-English input method.

修复拼音输入法下 TextInput maxLength 不能输入的 bug。

## Installation

```sh
npm install react-native-textinput-maxlength-fixed
```
